Today's jumble

Scrambled word game inspired by NewsRadio (S03EP04) and developed in JavaScript by ndzerglink.

Some words were scrapped from the Books: Pride and Prejudice by Jane Austen and Zealot by Reza Aslan.

Instructions:

Unscramble these Jumbles, placing one letter on each square, to form ordinary words.
Use DEL key to delete current input. Temporarily, use TAB key to input the next letter.
The letter and bottom bar are colored in Tomato when it is wrong, or in SlateBlue when correct.
This site is best viewed on Firefox
